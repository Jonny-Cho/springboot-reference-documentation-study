## 파일명 및 PR 규칙
1. 파일명 규칙
   - 형식 : 팀_이름_아이템내용
   - 예시 : B팀_최락준_Getting_Started.md
2. PR제목 규칙
   - 형식 : [팀][이름] 아이템내용
   - 예시 : [B팀][최락준] Getting Started
